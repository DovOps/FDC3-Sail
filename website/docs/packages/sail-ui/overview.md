# @finos/sail-ui

Shared React UI component library for FDC3 Sail. Built on [shadcn/ui](https://ui.shadcn.com/) and Tailwind CSS.

**Location:** `packages/sail-ui/`

Private monorepo package — not published to npm.

## Usage

```typescript
import { Button, Sidebar, SidebarProvider } from "@finos/sail-ui"
import "@finos/sail-ui/styles"
```

## Components

Layout: `Sidebar`, `SidebarContent`, `SidebarProvider`, `SidebarMenu`, …

Controls: `Button`, `DropdownMenu`, `Popover`, `Dialog`, `Tabs`, `Tooltip`, …

See the package source under `packages/sail-ui/src/components/` for the full list.
