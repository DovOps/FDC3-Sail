# @finos/sail-ui

Shared React UI component library for FDC3 Sail (shadcn/ui + Tailwind). Private monorepo package.

## Documentation

[finos.github.io/FDC3-Sail/docs/packages/sail-ui/overview](https://finos.github.io/FDC3-Sail/docs/packages/sail-ui/overview)

## Usage

```typescript
import { Button } from "@finos/sail-ui"
import "@finos/sail-ui/styles"
```
